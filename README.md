## Deploy en GitHub Pages

Este proyecto ya queda preparado para desplegarse con GitHub Pages usando GitHub Actions.

1. Sube estos cambios a la rama `main`.
2. En GitHub entra a `Settings` > `Pages`.
3. En `Build and deployment`, cambia `Source` a `GitHub Actions`.
4. Espera a que termine el workflow `Deploy to GitHub Pages`.
5. Tu sitio debería publicarse en:

`https://juandavid-sys.github.io/profile-Juan-Duarte/`

Si cambias el nombre del repositorio, la URL final también cambia.
